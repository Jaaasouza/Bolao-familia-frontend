# Archive

`usam-world-cup-2026.html` is a deprecated **standalone prototype** of the
pool — a single-file vanilla-JS app that:

- ran without any backend (all state in `localStorage`)
- called football-data.org directly from the browser
- used a hard-coded admin PIN

It predates the current React + Vite + Node/Postgres architecture (see
`../web/` and the companion backend repo `Bolao-familia-backend`).

**Don't edit it. Don't deploy it.** Kept here only for reference / for anyone
who once bookmarked the file and wants to see what they remember. All product
work happens in `../web/`.
